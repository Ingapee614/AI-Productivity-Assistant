# AI Workplace Productivity Assistant

## Project Overview

The AI Workplace Productivity Assistant is a modern web application designed to help professionals automate common workplace tasks using Artificial Intelligence. The platform combines multiple AI-powered tools into a single dashboard, enabling users to improve productivity, save time, and streamline daily workflows.

The application provides intelligent assistance for generating professional emails, summarizing meeting notes, planning schedules, conducting research, and interacting with an AI chatbot. The project demonstrates practical AI implementation, prompt engineering, responsive user interface design, and responsible AI practices.

---

## Features

### Smart Email Generator

* Generates professional emails based on user requirements.
* Supports multiple tones including:

  * Formal
  * Friendly
  * Professional
  * Persuasive
  * Apologetic
* Editable output before sending or copying.

### Meeting Notes Summarizer

* Summarizes lengthy meeting notes and transcripts.
* Extracts:

  * Key discussion points
  * Decisions made
  * Action items
  * Deadlines

### AI Task Planner

* Creates optimized daily or weekly schedules.
* Prioritizes tasks based on urgency and importance.
* Helps users manage workload efficiently.

### AI Research Assistant

* Summarizes topics and articles.
* Provides:

  * Key insights
  * Advantages
  * Challenges
  * Recommendations

### AI Chatbot Interface

* Interactive workplace assistant.
* Answers productivity-related questions.
* Assists with drafting content, organizing tasks, and providing information.

### Responsive Dashboard

* Modern SaaS-style user interface.
* Sidebar navigation for easy access to tools.
* Responsive design for desktop, tablet, and mobile devices.

### Responsible AI Disclaimer

* Informs users that AI-generated content should be reviewed before use.
* Promotes ethical and responsible AI usage.

## Setup Instructions

To run the AI Workplace Productivity Assistant locally, first ensure that Node.js, npm, and Git are installed on your computer. Clone the project repository from GitHub and navigate to the project folder using your terminal or command prompt. Install all required dependencies by running the `npm install` command. Next, create a `.env` file in the root directory of the project and add your OpenAI API key to enable AI-powered features. Once the configuration is complete, start the development server using the `npm run dev` command. The application will launch locally, and you can access it through your web browser at the URL provided in the terminal, typically `http://localhost:5173`. The dashboard will then be available for testing and use, allowing you to access all AI-powered productivity features.
